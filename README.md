# `astroforge_celestial`
Celestial mechanics in Rust.

# Installation

## From source

1. Clone repository:

````
git clone https://github.com/astro-forge/celestial.git
````

2. Enter the source root:

````
cd celestial
````

3. Build (requires [Rust toolchain](https://www.rust-lang.org/tools/install)):

````
cargo build
````
